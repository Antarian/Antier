Antier CMS
===

Just another PHP CMS or JS framework?


Docker compose
--------------

Docker compose build and start
```bash
docker-compose build
docker-compose up -d
```

Docker VM
```bash
docker-compose exec antier bash
```

Symfony
---

Server start
```bash
php bin/console server:start *:8080
```

In browser
```
0.0.0.0:8080
```

Tests
```bash
./bin/phpunit
```
